# RL Lab 7: MENACE & Multi-Armed Bandits

Reinforcement Learning implementations for MENACE (Tic-Tac-Toe) and Multi-Armed Bandit problems.

## Structure

```
├── src/          # Python implementations
├── results/      # Generated plots
└── docs/         # LaTeX report
```

## Usage

```bash
pip3 install matplotlib numpy
python3 src/ex7_1.py  # MENACE
python3 src/ex7_2.py  # Binary Bandit
python3 src/ex7_3.py  # Non-Stationary Bandit
python3 src/ex7_4.py  # Agent Comparison
```
